# ☕ Excel Coffee Sales Dashboard – Guided Learning Project

## 📌 Project Overview

This project was completed as a guided learning exercise while developing my Excel and Data Analytics skills.

By following an Excel dashboard tutorial, I recreated the project from scratch to gain practical experience in data preparation, data analysis and dashboard development. The goal was to understand how raw business data can be transformed into meaningful insights through interactive reporting.

The dashboard analyses coffee sales performance across products, customers, and countries, allowing users to explore trends and patterns through dynamic filters and visualisations.

---

## 🎯 Learning Objectives

Through this project, I practiced:

* Data Cleaning and Preparation
* Excel Lookup Functions
* Data Transformation
* Pivot Tables
* Pivot Charts
* Interactive Dashboard Design
* Slicers and Timeline Filters
* Business Reporting and Analysis

---

## 🛠️ Project Workflow

### Data Preparation

* Combined data from Orders, Customers, and Products tables.
* Retrieved customer information using **XLOOKUP**.
* Retrieved product information using **INDEX-MATCH**.
* Created calculated fields such as Sales.

### Data Cleaning & Transformation

* Standardized and formatted data fields.
* Applied custom formatting for dates, currency, and product information.
* Removed inconsistencies and prepared data for analysis.
* Converted data into Excel Tables for easier management and reporting.

### Data Analysis

* Built Pivot Tables to analyse sales performance.
* Examined customer behaviour and purchasing trends.
* Compared sales across different countries and product categories.

### Dashboard Development

* Created Pivot Charts to visualise key metrics.
* Added Timeline and Slicer controls for dynamic filtering.
* Connected dashboard elements using Report Connections.
* Designed a single-page interactive dashboard for reporting and exploration.

---

## 📊 Dashboard Features

### Sales Trend Analysis

* Track total sales over time.
* Compare performance across different coffee types.

### Country-wise Sales Analysis

* Compare sales across:

  * United States
  * United Kingdom
  * Ireland

### Customer Analysis

* Identify top-performing customers based on sales value.

### Interactive Filtering

Users can dynamically explore data using:

* Timeline (Date Filter)
* Roast Type
* Product Size
* Loyalty Card Status
