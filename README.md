# Excel Coffee Sales Dashboard

## Overview

This project is an interactive Coffee Sales Dashboard built using Microsoft Excel.

The dashboard provides insights into sales trends, customer performance, product categories and country-wise sales through interactive visualisations.

## Tools & Features Used

- Microsoft Excel
- XLOOKUP
- INDEX MATCH
- IF Functions
- Pivot Tables
- Pivot Charts
- Slicers
- Timeline Filters
- Data Cleaning
- Data Formatting

## Business Questions Answered

- How are sales trending over time?
- Which countries generate the highest sales?
- Who are the top 5 customers?
- How do different coffee types perform?
- Does loyalty card status impact sales?
